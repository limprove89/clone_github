#### meta

- name과 content로 사이트 전반적인 정보를 기입
- og(open graph)/ twitter card의 상세 기입

#### link

- icon -> png파일로 연결
- apple-touch-icon 은 ios/Android에서 앱 화면 설정
- index.html 위치 내 favicon.ico 파일이 위치되어있으면, 구형 브라우저에서도 찾아서 인식
- font cdn과 reset.css, 작성하는 css파일을 연결
